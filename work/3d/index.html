This project uses a 2D turtle graphics system to render 3D objects. A system of classes representing 3D points, polygons and solids was created to facilitate work with these structures. The original 2D Turtle class was extended with functions accepting these 3D Objects. Upon startup, the document class procedurally generates a unique fracture pattern for the glass, and gives it to the turtle for rendering.

On every frame of the animation, a rotation matrix is applied to slowly turn the model. The shading for each polygon is also recalculated to allow for a slight glint in the glass. This was done by computing the angle between each polygon's surface normal and the camera's forward vector. When drawing, the system also interpolates between vertices to allow for the slight curved lens distortion.
