Interactive Optics Book | Flash ActionScript 3

This project is based on an excerpt from the book "How It Works" by Archibald Williams. The goal was to make the book interactive and fun for the user, while still providing all of the text from the original book.

Challenges:

  • Computing Lens/Ray intersections
  • Implementing Snells law
  • Interface structure and implementation
