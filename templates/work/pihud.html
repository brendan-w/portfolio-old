This is a heads-up display for a car, built to run on a Raspberry Pi. It uses my Python OBD-II serial module to stream live data from the car. The software autodetects available sensors within the car, and provides a primitive interface for users to design their HUD. Widgets representing the various sensors can be added, deleted, positioned, and resized to the drivers liking.
