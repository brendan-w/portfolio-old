C-Graph is a function call graphing utility for C with GitHub integration. Simply provide the url for a Git repository, and C-Graph will search for C files. The graph itself represents the code at both the file and function level. Files can be expanded into sets of functions, with arrows depicting the calls from one function to the next. Selecting a function or file will also display its raw C code.
