3D Turtle Graphics | Flash ActionScript 3

This projects uses a 2D turtle graphics system to render 3D objects. A system of classes representing 3D Points, Polygons, and Objects was created to facilitate work with these structures. The original 2D Turtle class was extended with functions accepting these 3D Objects. Upon startup, the document class procedurally generates a unique fracture pattern for the glass, and gives it to the turtle for rendering.

Challenges:

  • Projection of 3D space onto 2D screen
  • Computing surface normals
  • Shading mathematics
  • Object occlusion logic for bullet
