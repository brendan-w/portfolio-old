C-Graph is a function call graphing utility for C. Integrated with GitHub, C-Graph contructs a force directed graph of function and files in a Git repository. Files can be expanded into sets of functions, with arrows depicting the calls from one function to the next. Selecting a function or file will also display its raw C code.
