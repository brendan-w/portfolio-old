This program was inspired by the physical spirograph drawing toy. However, instead of having a limited selection of radii and patterns, this application allows the user to explore the full range of the mechanism. Because of this, the application can produce much more complex patterns than its geared counterpart. While most XY coordinates produce dense rings of lines, there are still many sweet spots where the typical spirograph patterns can be found.

Spirograph Generator Processing
Github Code

Internally, the mouse position is used to adjust the proportions of the spirograph mechanism. The equations then generate a sequence of points. These points are sequentially connected with short line segments to form a curve. After every revolution, the application checks to see if the latest point has returned to the initial starting point. If so, it is safe to assume that the curve has reached its cycle, and will only repeat itself if the process continues. However, in the event that any particular equation has an excessively long cycle, the application stops after 250 revolutions.
