An application developed for the OLPC aimed to teach 4th graders about the planets and our solar system. With a quiz-based system, facts about the planets are gradually dispensed to the user as they answer questions correctly. This prevents "information overload," and ensure that the user has learned the existing content before advancing to more difficult concepts. Written with Python and Pygame, this application runs both on the OLPC, and as a standalone Pygame app.
