Collector is a tag-based file manager for linux. It presents files in a flat data structure where the user can select files by tag names.

Collector stores tags in the file's name and path. This allows directory names to function as tags, making existing folder structures easier to work with. When tags are added or removed, Collector will either splice the tag in to or out of the filename, or move the file to a different spot in the directory tree. Not only does this help maintain an organized directory structure, but also prevents the need to synchronize a tag database with the filesystem.
