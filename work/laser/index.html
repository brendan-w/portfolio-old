This program was built to control a laser graphics projector. Like the lasers used at rock concerts, the hardware uses vector graphics to scan/project any programmed image. This software provides an interface to generate these graphics. Tools such as point manipulation, interpolation, and effects are all built in to the software. Once created, the image can then be sent to the projector via that parallel port.

Laser Controller Visual Basic

This program also features an effects stack. This lets the user sequentially apply modifiers to the drawn image. Effects such as translation, scale, and duplication (both linear and radial) can be applied to produce the desired effect. Some of these effects even provide rudimentary animation capabilities.
