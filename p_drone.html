Drone Game | Flash ActionScript 3

This flash game was done as a final project for RIT's Intro to Programming course. It features four levels in which the user can pilot a quadrotor drone around a course filled with enemy turrets. By using any of the three weapon types, the user can clear the playing field of all turrets.

Challenges:

  • Line-of-sight computation
  • Movement & recoil physics
  • Interface structure and implementation
