This flash game was done as a final project for RIT's Intro to Programming course. It features four levels in which the user can pilot a quadrotor drone around a course filled with enemy turrets. By using any of the three weapon types, the user can clear the playing field of all turrets. The two main developement challenges for this game were line-of-sight computation, and movement/recoil physics.
