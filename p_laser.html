Laser Controller | MS Visual Basic 6

This program was built to control a laser graphics projector, like those used at rock concerts. On top of handling the vector laser graphics, this program also features an effects stack. This lets the user sequentially apply modifiers to the drawn image. Once computed, the program then sends the data to the projector via that parallel port.

Challenges:

  • Point transformations
  • Animation engine
  • Hardware interaction
  • Interpolation routine
