One of my many hobbies is building laser projectors. Much like those found at rock concerts, this projector is capable of receiving and drawing vector graphics. The projector shown was built back in highschool.

In order to control the projector, I wrote custom software in visual basic (I was a young programmer). This provided an interface in which to draw vector laser. Tools such as point manipulation, interpolation, and effects are all built in to the software. Once drawn, the image can then be sent to the projector via that parallel port.

This program also features an effects stack. This lets the user sequentially apply modifiers to the drawn image. Effects such as translation, scale, and duplication (both linear and radial) can be applied to produce the desired effect. Some of these effects even provide rudimentary animation capabilities.
